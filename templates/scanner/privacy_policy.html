Information We Collect

ZtionSec is committed to protecting your privacy. We collect information to provide better security analysis services:

Information You Provide
  • Website URLs: URLs you submit for security scanning
  • Email Addresses: For breach checking and newsletter subscriptions
  • Contact Information: When you contact our support team
  • Account Data: If you create an account (optional)
Automatically Collected Information
  • Technical Data: IP address, browser type, device information
  • Usage Data: Pages visited, scan history, feature usage
  • Security Scan Results: Technical findings from security assessments
  • Performance Metrics: Response times, error rates, system performance

How We Use Your Data

We use collected information for legitimate business purposes:

Service Provision
  • Perform security scans and analysis
  • Generate security reports
  • Provide vulnerability assessments
  • Deliver breach notifications
Service Improvement
  • Analyze platform performance
  • Improve security detection algorithms
  • Enhance user experience
  • Develop new security features
No Personal Data in Scans: We only analyze publicly available information from websites. We do not access private or sensitive user data during security scans.

Data Protection & Security

We implement industry-standard security measures to protect your information:

Encryption

All data transmitted using SSL/TLS encryption

Secure Storage

Data stored on secure, monitored servers

Access Control

Limited access on need-to-know basis

Data Retention
  • Scan Results: Retained for 90 days for analysis and reporting
  • Email Addresses: Retained until you unsubscribe
  • Technical Logs: Retained for 30 days for security monitoring
  • Account Data: Retained while account is active
