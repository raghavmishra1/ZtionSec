ZtionSec Documentation

Complete guide to using ZtionSec security analysis platform

Getting Started

ZtionSec is a comprehensive security analysis platform that helps you identify vulnerabilities and security issues in websites and web applications.

Quick Start

  1. Enter a website URL in the scanner
  2. Choose your scan type (Basic, Advanced, or Budget)
  3. Review the security report
  4. Follow remediation recommendations

Website Scanner

The basic website scanner performs comprehensive security analysis including:

  • SSL/TLS Analysis - Certificate validation and grading
  • Security Headers - HSTS, CSP, X-Frame-Options, etc.
  • Performance Testing - Response time analysis
  • Technology Detection - CMS and framework identification

Advanced Scanning

Advanced scanning provides enterprise-grade security analysis:

  • Vulnerability Assessment - OWASP Top 10 and beyond
  • Network Analysis - Port scanning and service detection
  • DNS Intelligence - Subdomain enumeration and DNS analysis
  • Threat Intelligence - CVE correlation and threat analysis

Budget Scanner (P4 Vulnerabilities)

Focused on easy-to-find, low-severity vulnerabilities perfect for bug bounty hunters:

  • Information Disclosure - Config files, backups, debug info
  • Misconfigurations - Directory listings, exposed files
  • Low-Hanging Fruit - Quick wins with minimal effort
  • Bounty Potential - Estimated reward ranges

API Reference

ZtionSec provides RESTful APIs for integration:

Endpoints

Endpoint Method Description
/api/v1/scan/ POST Start basic security scan
/api/v1/advanced-scan/ POST Start advanced security scan
/api/v1/budget-scan/ POST Start budget security scan
/api/v1/vulnerabilities/ GET Get vulnerability database

Enterprise Features

Enterprise customers get additional features:

  • Team Management - Multi-user access and permissions
  • Custom Reports - Branded PDF reports and dashboards
  • API Access - Full API access with higher rate limits
  • Priority Support - Dedicated support and SLA
  • Compliance Reporting - OWASP, NIST, ISO 27001 reports
